LibRobot 

- Pour la documentation du robot, veuillez vous referer au 
  document "documentation.pdf" pour la documentation en version
  compact sur les fonctions et la librairie. Sinon une documentation 
  plus complete en html se retrouve dans le dossier "Docs/html/index.html".

- Pour observer des cas d'utilisation, veuillez voir le dossier
  tests, il y a un dossier pour tous les modules dans tests..

- Le logiciel du chat et de la souris se retrouve egalement dans le
  dossier tests.
  
- La liste des pins utilisees sont specifiees dans le document 
  "AssignationBroches.txt".
  
- Pour le fonctionnement du Makefile, veuillez vous referer au docuement
  "usage du makefile.txt".
